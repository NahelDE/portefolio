Soûl-Mates

Le but de ce projet était de créer un quizz permettant de deviner les ingrédients d'un cocktail. Nous avons utilisé un workflow "developer branch" : chacun travaillait sur sa branche et fusionnait sur la branche dev après validation. Le projet a été réalisé avec Flask et Jinja2, intégrant une API et des fonctionnalités collaboratives. J’ai beaucoup progressé en gestion de projet, c’était la première fois que j’utilisais la méthode agile. De plus, j’étais Scrum Master et avais donc la responsabilité du projet. Ce fut une expérience très enrichissante.

Compétences : Python, Flask, Jinja2, API, Git

Page d'accueil du quizz, interface principale visible pour les utilisateurs.

Aperçu du quiz où l'utilisateur doit deviner le cocktail.

Extrait de la fonction save_score_and_redirect(name_form) qui enregistre le score dans la BDD et redirige.

Lensymphony & Web Lensymphony

Lensymphony est un projet ambitieux et créatif consistant à développer un synthétiseur de musique en Java, capable d'imiter différents instruments grâce à des formules mathématiques et des patrons de conception. Nous avons codé de nombreux instruments, géré plusieurs portées simultanées et lu des partitions XML pour reproduire des morceaux complexes.

Le projet est double : Lensymphony, le synthétiseur Java, et Web Lensymphony, un site web en PHP/Blade/HTML qui permet de générer et d’écouter de la musique via le synthétiseur. Ce mélange de backend, frontend et programmation Java a rendu le projet à la fois stimulant et très formateur.

Compétences : Java, Gradle, PHP, Blade, HTML, Git

Page d'accueil du site, permettant d’accéder aux fonctionnalités du synthétiseur.

Page affichant les partitions XML pour la lecture et la génération musicale.

Extrait de la fonction synthesize() qui génère le son d’un kickdrum, imitant l’instrument en Java.

Casino JavaFX

Ce projet personnel consiste en une application de casino développée en Java avec JavaFX, proposant plusieurs jeux dont le Blackjack et le Pile ou Face. Réalisé entièrement en solo, ce projet m’a permis de travailler sur la logique de jeu, la gestion d’un solde joueur et la création d’interfaces graphiques interactives.

Compétences : Java, JavaFX

Interface graphique du jeu de Blackjack développée en JavaFX.

RogueAbyss

RogueAbyss est un projet personnel de roguelike 2D développé en Java avec Swing. Le jeu est encore en phase de développement early, permettant actuellement de se déplacer sur la map générée et de combattre des ennemis simples (des carrés rouges).

La map est générée aléatoirement à chaque partie, et certaines salles prédéfinies sont reliées par des couloirs. Le gameplay roguelike complet n’est pas encore implémenté, mais le projet m’a déjà permis de travailler sur la POO, la génération procédurale et la gestion d’éléments graphiques avec Swing.

Compétences : Java, Swing

Combat basique contre un carré rouge représentant un ennemi.

Simple Currency Exchange

Ce projet personnel consiste en une application web développée avec Flask, permettant de convertir des montants d’une devise à une autre en utilisant une API externe.

Actuellement, la fonction all_currency_in_euro() récupère les taux de change par rapport à l’euro et les affiche sous forme lisible pour l’utilisateur. Par exemple, on peut voir combien vaut le dollar en euro, avec des taux à jour grâce à l’API. L’interface utilisateur et certaines fonctionnalités comme choisir la devise de départ et d’arrivée sont encore en cours de développement.

Compétences : Python, Flask, HTML/Template Jinja2

Exemple de conversion : affichage des taux en euro pour chaque devise.

Extrait du code Flask qui récupère les taux via l’API et les affiche dans la page web.
