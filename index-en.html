Lensymphony & Web Lensymphony

Lensymphony is an ambitious and creative project developing a music synthesizer in Java, capable of imitating various instruments using mathematical formulas and design patterns. We coded multiple instruments, handled simultaneous musical staffs, and read XML scores to reproduce complex pieces.

The project is dual: Lensymphony, the Java synthesizer, and Web Lensymphony, a PHP/Blade/HTML website to generate and listen to music via the synthesizer. This combination of backend, frontend, and Java programming made the project both challenging and highly educational.

Skills: Java, Gradle, PHP, Blade, HTML, Git

Homepage of the site providing access to the synthesizer functionalities.

Page displaying XML scores for musical playback and generation.

Snippet of the synthesize() function generating a kick drum sound, imitating the instrument in Java.

RogueAbyss

RogueAbyss is a personal 2D roguelike project developed in Java with Swing. The game is still in early development, currently allowing movement on the generated map and combat with simple enemies (red squares).

The map is randomly generated each game, and some predefined rooms are connected by corridors. The full roguelike gameplay is not yet implemented, but the project has already allowed me to work on OOP, procedural generation, and graphical element management with Swing.

Simple Currency Exchange

This personal project is a web application developed with Flask, allowing conversion of amounts from one currency to another using an external API.

Currently, the function all_currency_in_euro() retrieves exchange rates relative to the Euro and displays them in a user-friendly format. For example, it shows how much a dollar is worth in euros, with up-to-date rates thanks to the API. The user interface and some features, such as choosing the source and target currencies, are still under development.

Skills: Python, Flask, HTML/Jinja2 Templates
